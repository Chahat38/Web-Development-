# Web-Development-
  # HTML, CSS and JavaScript Practice Projects

This repository contains simple web development practice projects created using **HTML and CSS**.
The purpose of this repository is to practice basic web design concepts such as **forms, navigation bars, audio/video embedding, and page layouts**.

## 📂 Project Files

### 1. Appointment Form

**File:** `Appointment P...`
A simple appointment booking form created using HTML form elements such as:

* Name
* Email
* Date
* Message
* Submit button

This project helps understand **HTML forms and input fields**.

### 2. Audio Page

**File:** `Audio.html`
This page demonstrates how to embed **audio in HTML** using the `<audio>` tag and controls.

### 3. Left Navigation Bar

**File:** `Leftbar.html`
This page contains a **vertical navigation bar displayed on the left side of the page** using CSS layout.

### 4. Navbar Page

**File:** `Navbar.html`
This page demonstrates how to create a **top navigation bar** using HTML and CSS.

### 5. Login Page

**File:** `LoginPage.html`
A simple **login form** with fields such as:

* Username
* Password
* Login button

### 6. Signup Page

**File:** `Signup.html`
A **registration form** where users can create an account by entering their information.

### 7. Links and Video Page

**File:** `linksvideo.html`
This page shows how to use:

* Anchor tags (`<a>`)
* Embedded videos using `<iframe>`.

### 8. CSS Files

* `style.css`
* `style2.css`
* `leftnavstyle.css`

These files are used to apply **styling to different HTML pages** such as layout, colors, navigation bars, and forms.

## 🎯 Purpose of This Repository

* Practice **basic HTML elements**
* Learn **CSS styling**
* Understand **web page structure**
* Build beginner-level web pages

## 🛠 Technologies Used

* HTML5
* CSS3
